# 💊 AI-Based Drug Repurposing Prediction Model

This project leverages Machine Learning to identify new therapeutic uses for existing drugs. By analyzing drug features and biological interactions, the model predicts potential drug-disease associations, accelerating the drug discovery process while reducing cost and time.

---

## 🚀 Project Overview

Drug repurposing (or drug repositioning) is a strategy to discover new uses for approved or investigational drugs. This project applies AI/ML techniques to predict possible new indications for existing drugs using computational approaches.

---

## 🔬 Key Features

- ✔ Predicts potential new drug-disease associations  
- ✔ Uses molecular and/or biological feature representations  
- ✔ Machine Learning-based prediction model  
- ✔ Data preprocessing and feature engineering pipeline  
- ✔ Scalable and modular code structure  

---

## 🧠 Tech Stack

- **Programming Language:** Python  
- **Libraries:** Pandas, NumPy, Scikit-learn  
- **Cheminformatics (optional):** RDKit  
- **Model Saving:** Joblib  
- **Interface (optional):** Streamlit  

---

## ⚙️ Workflow

### 1️⃣ Data Collection
- Drug-related data (chemical, biological, or clinical)
- Drug-disease association dataset  

### 2️⃣ Data Preprocessing
- Handling missing values  
- Encoding categorical variables  
- Data normalization  

### 3️⃣ Feature Engineering
- Molecular descriptors / fingerprints (RDKit)  
- Interaction-based or statistical features  

### 4️⃣ Model Building
- Algorithms used:
  - Random Forest  
  - Logistic Regression  
  - Support Vector Machine (SVM)  

### 5️⃣ Model Evaluation
- Accuracy  
- Precision  
- Recall  
- F1-score  

### 6️⃣ Prediction
- Input: Drug features  
- Output: Probability of new disease indication
